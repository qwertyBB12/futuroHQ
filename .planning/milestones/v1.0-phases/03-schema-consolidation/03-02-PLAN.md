
   ```
   defineField({
     name: 'linkedVideo',
     title: 'Linked Video',
     type: 'reference',
     to: [{ type: 'video' }],
     description: 'Video recording of this keynote speech',
   })
   ```

3. **linkedEssay** — Reference to published essay version:
   ```
   defineField({
     name: 'linkedEssay',
     title: 'Linked Essay',
     type: 'reference',
     to: [{ type: 'essay' }],
     description: 'Published essay version of this keynote, if one exists',
   })
   ```

4. **linkedPodcastEpisode** — Reference to podcast version:

   ```
   defineField({
     name: 'linkedPodcastEpisode',
     title: 'Linked Podcast Episode',
   ```
