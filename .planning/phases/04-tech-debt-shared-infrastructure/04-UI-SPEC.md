## Spacing Scale

Declared values (must be multiples of 4). Sanity's internal spacing system uses 4px base increments. Custom components and injected styles follow the same scale.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge padding, inline element gaps |
| sm | 8px | Compact element spacing, list item margins (`margin: 2px 6px` on PaneItems) |
| md | 16px | Default element spacing, field group padding |
| lg | 24px | Section padding, field group separation |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Dashboard widget internal padding |
| 3xl | 64px | Page-level spacing, dashboard row gaps |

Exceptions: PaneItem margins use `2px 6px` (non-standard) — this is an existing override in `styles.css` and must not be changed in this phase.

## Typography

All type is rendered by `@sanity/ui` internals except custom components (MyNavbar, dashboard widgets, badge labels). The following scale applies to any custom-rendered text in this phase.

| Role | Font | Size | Weight | Line Height | Source |
|------|------|------|--------|-------------|--------|
| Body / UI labels | Mulish | 14px | 400 (regular) | 1.5 | CLAUDE.md, styles.css |

| Section heading | Mulish | 16px | 600 (semibold) | 1.4 | default |
| Brand / navbar title | Oswald | 20px | 700 (bold) | 1.2 | CLAUDE.md, MyNavbar.tsx pattern |

Weights used: **400 (regular)** and **600 (semibold)**. No other weights in custom UI surfaces this phase.
